# Tutorial Builder UI

A simple Minecraft-style website for creating step-by-step tutorial JSON files.

## Features
- Add steps with SPACE or ENTER
- Reorder steps
- Delete steps
- Export JSON
- Minecraft-style font

## How to use
1. Paste your YouTube link
2. Click "Start Tutorial"
3. Watch the video in another tab
4. Press SPACE or ENTER to add steps
5. Type step text
6. Export JSON
